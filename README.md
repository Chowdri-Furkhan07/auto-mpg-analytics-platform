# 📌 Overview

Auto MPG Analytics Platform is an end-to-end **Data Analytics & Business Intelligence solution** designed to analyze vehicle fuel efficiency, engine performance, and automotive market trends using the renowned **Auto MPG Dataset**.

This project combines:

* 📊 Exploratory Data Analysis (EDA)
* 📈 Interactive Business Intelligence Dashboards
* ⚡ Real-Time KPI Monitoring
* 🎯 Executive-Level Insights
* 🌐 Web-Based Analytics Applications

The platform enables users to uncover insights related to fuel efficiency, horsepower impact, vehicle weight distribution, and regional automotive trends through highly interactive dashboards.

---

# 🎯 Business Problem

Automotive manufacturers and analysts require data-driven insights to understand:

* Fuel efficiency trends across decades
* Engine performance impact on mileage
* Vehicle weight optimization
* Regional market comparisons
* Consumer efficiency patterns

This platform provides analytical dashboards and visualization systems that transform raw automotive data into actionable business insights.

# ▶️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Chowdri-Furkhan07/auto-mpg-analytics-platform.git
```

---

## 2️⃣ Navigate to Project Directory

```bash
cd auto-mpg-analytics-platform
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Applications

## Streamlit Dashboard

```bash
streamlit run app.py
```

## Dash Dashboard

```bash
python dash_app.py
```
